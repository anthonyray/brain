Little Brain

The project

Little brain is a simple implementation of a multi layer neural network using backpropagation algorithm in javascript. The neural network was trained before being put online. This neural network is able to recognize digits between 0 and 3 drawn on a 4 x 5 pixels matrix.

About the neural network

You can find the code on my github profile : here. Don't hesitate to fork, make a pull request, or open an issue !

This neural network is made of two layers : one input layer and one output layer.

The input layer has 20 neurons. Each neuron correspond to one pixel of the drawing matrix.

The output layer has 2 neurons. We want our neural network to predict a number between 0 and 3. If we consider the output of each neuron as a bit value, the output layer produces a number between 0 and 3.

The neural network was trained using the backpropagation algorithm.
